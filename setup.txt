setting up new server on EC2Spot instance
1) sudo yum install nodejs npm --enablerepo=epel
2) Enable inbound traffic on 80 and 22 ports for the security group
3) sudo npm install googleapis --save
4) npm install open --save
5) npm install express --save
6) npm install body-parser --save
7) npm install aws-sdk
